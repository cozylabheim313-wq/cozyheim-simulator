# Cozyheim Revenue Simulator

방콕 야오와랏 호스텔 수익 시뮬레이터 PWA

## 배포 방법 (5분)

### 1. GitHub 계정 만들기 (이미 있으면 skip)
- github.com 접속 → Sign Up

### 2. Repository 만들기
- 우측 상단 + 버튼 → "New repository"
- Repository name: `cozyheim-simulator`
- Public 선택
- "Create repository" 클릭

### 3. 파일 업로드
- "uploading an existing file" 클릭
- 이 폴더 안의 파일 전체 드래그 (index.html, manifest.json, sw.js, icons 폴더)
- "Commit changes" 클릭

### 4. GitHub Pages 활성화
- Repository → Settings → Pages
- Source: "Deploy from a branch"
- Branch: main / (root)
- Save

### 5. 완료
- 약 1분 후 `https://[계정명].github.io/cozyheim-simulator` 접속 가능
- 이 URL을 카카오톡으로 공유

## 홈 화면 추가 방법 (직원 안내)

**iPhone:**
Safari로 열기 → 하단 공유 버튼 → "홈 화면에 추가"

**Android:**
Chrome으로 열기 → 주소창 옆 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
